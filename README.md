# 99blog

99blog 是赵春昊（清9半斛）的个人博客与作品展示站。

这里会持续沉淀关于 AI 原生创业、Agent 开发、产品思考、项目复盘和个人成长的内容，也会展示正在打磨的作品、经历和联系方式。

## 你可以在这里看到什么

- **文章**：围绕 AI、产品、创业和技术实践的长期记录。
- **项目**：展示个人作品、产品实验和阶段性成果。
- **经历**：整理校园、比赛、实践、创业和工作相关经历。
- **关于我**：了解赵春昊（清9半斛）的方向、技能和关注主题。
- **联系入口**：找到 GitHub、个人站、微信等外部入口；暂未补齐的链接会显示为“待补充”。

## 页面入口

| 页面 | 路径 | 说明 |
|---|---|---|
| 首页 | `/` | 快速了解个人定位、重点内容和核心入口。 |
| 文章 | `/posts` | 浏览文章列表，支持按关键词和分类筛选。 |
| 项目 | `/projects` | 查看项目作品、技术栈、状态和相关链接。 |
| 经历 | `/experience` | 查看经历时间线、组织、时间段和亮点。 |
| 关于 | `/about` | 查看个人介绍、头像、技能方向和一句话定位。 |
| 联系 | `/contact` | 查看联系方式和外部入口。 |

## 内容方向

99blog 当前重点关注这些主题：

1. **AI 原生创业**：记录从想法、MVP 到产品化的探索。
2. **Agent 开发**：沉淀智能体、自动化工作流和工具实践。
3. **产品思考**：复盘需求、体验、增长和长期价值。
4. **项目作品**：展示从 0 到 1 的构建过程和结果。
5. **个人成长**：记录学习、比赛、实践与阶段性总结。

## 阅读建议

如果你是第一次访问，可以按这个顺序浏览：

1. 先看首页，了解整体风格和核心入口。
2. 再看项目页，了解正在做什么。
3. 然后看文章页，阅读更完整的思考和复盘。
4. 最后看关于页和联系页，了解更多背景与联系方式。
